Hospedagem

O que é?

A hospedagem de um site é oque possibilita você a publicá-lo na internet.Os serviços de hospedagem guardam todos os seus dados do site para tornar acessível aos possíveis usuários.

O que é necessário para hospedar um software?

Os passos fundamentais para você começar a hospedar são :

  • Registrar um domínio
  • Escolher um serviço de hospedagem
  • Configurar o servidor de DNS
  • Configurar o registro de DNS
  • Configurar o servidor de hospedagem
  • Configurar as contas de E-Mail
  • Transferir seu site para o servidor de hospedagem

DNS

O DNS vem de Domain Name System(Sistema de Nomes de Domínio),ele é oque lhe permite a entrar em um site pelo seu nome de domínio na barra de pesquisa.Por exemplo,quando digitamos google.com.br.

SEO

O SEO vem de Search Engine Optimization(otimização de mecanismos de pesquisa),é a prática e a técnica de se otimizar um site para deixá-lo no topo ou mais visível para os mecanismos de pesquisa.
