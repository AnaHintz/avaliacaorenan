Git e Github

Git

O Git consiste em uma ferramenta de versionamento local,que salva o estado de cada arquivo na hora do versionamento e caso o arquivo não sofra a alteração ele cria um link simbólico para o arquivo que não foi editado.

Github

O Github é um serviço web compartilhado para projetos que utilizam a ferramenta do Git para realizae o versionamento.É um local para armazenar códigos.

LEMBRE-SE :

O Git funciona sem o Github,já o Github NÃO funciona sem o Git

Como configurá-los

Após criar sua conta no Github deverá configurá-lo com a chave SSH

  • Dentro do Git verifique se existe a chave SSH : ls -al ~/.ssh
  • Adicione uma nova chave,colocando seu E-Mail : (ID)ssh-keygen -t ed25519 -C "seu_email@exemplo.com"
  • Inicialize o agente-ss : eval "$(ssh-agent -s)"
  • Adicione chave ssh ao agente : ssh-add ~/.ssh/id_ed25519
  • Copie a chave ssh : clip < ~/.ssh/id_ed25519.pub
  • Adicionar chave no github : Github > settings > SSH and GPG keys > New SSH key > Colar *Coloque u título que identifique a chave*
  • Teste a conexão : ssh -T git@github.com > yes

Como criar um repositório e como cloná-lo

Para colocar seu projeto dentro do Github você precisará criar um repositório no qual ele ficará armazenado.

  • Para isso,dentro do Github basta clicar no canto superior da página onde terá um ícone de "+" e selecione "Novo repositório";
  • Depois didgte o nome para o seu repositório e crie uma breve descrição;
  • Escolha a visibilidade dele(se ele vai ser público ou privado);
  • Selecione Inicializar este repositório com um README;
  • Clique em Criar repositório.

Dentro do Github é possível clonar seu repositório para criar uma cópia local no seu computador e sincronizar entre os dois locais.

  • Dentro do Github,vá até a página principal do repositório;
  • Acima da lista de arquivos, clique em "< >Código";
  • Copie a URL do repositório.
  • Abra Git Bash e altere o diretório de trabalho atual para o local em que deseja ter o diretório clonado.Para isso digite "git clone" e a url copiado ao lado;
  • Pressione ENTER para criar seu clone local.
