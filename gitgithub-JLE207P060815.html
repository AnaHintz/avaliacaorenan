Git e Github

Git

O Git consiste em uma ferramenta de versionamento local,que salva o estado de cada arquivo na hora do versionamento e caso o arquivo não sofra a alteração ele cria um link simbólico para o arquivo que não foi editado.

Github

O Github é um serviço web compartilhado para projetos que utilizam a ferramenta do Git para realizae o versionamento.É um local para armazenar códigos.

LEMBRE-SE :

O Git funciona sem o Github,já o Github NÃO funciona sem o Git.
